Authors: Gunes Guner1; Margaret Cocks1; Diana Taheri1; Mark W. Ball2; Stephania M. Bezerra1; Alan Meeker1; Maria del Carmen Rodriguez1; Alcides Chaux1,4; Arthur Burnett1; George Netto5;

Material and method

Cohort features

The dataset included clinicopathologic and outcome data from 78 cases with papillary renal cell carcinoma. Using these cases, 2 tissue microarrays (TMA) were built, including 285 TMA spots. All the TMA spots included samples from nontumor and primary tumor tissues. Additional, 69 TMA spots included paired metastatic tumor tissues.

 

Scoring system

For each marker, the methodology for evaluation of the immunohistochemical expression was as follows:

  • PDL1: PDL1 was evaluated in tumor and stromal cells, noting the percentage of positive cells.
  • FOXP3, CD8, and Ki67: The expression of these markers was evaluated in lymphocytes infiltrating tumor and stroma, counting the number of positive lymphocytes.

Additionally, we created a 2-level category of the CD8-to-FOXP3 balance comparing the counts of positive cells in tumor and in stroma, as follows: CD8 counts greater than FOXP3 counts vs. CD8 counts less than or equal to FOXP3 counts.

Data analysis

Data analysis was carried out using 3 approaches: descriptive analysis, association analysis, and outcome analysis. Data was analyzed and plots were generated using R version 3.3.2 (2016-10-31), available at https://www.r-project.org/. The following additional R packages were esential for data analysis: dplyr, ggplot2, ggfortify, tidyr, forcats, and knitr.

 

Descriptive analysis

Categorical variables were described using frequency tables and barplots. Numerical variables were described using measurements of central tendency and dispersion, histograms, and density plots.

 

Association analysis

Association analysis included the comparison of biomarkers expression levels by tissue type (nontumor vs. primary tumor vs. metastatic tumor) and by clinicopathologic and outcome features. In this context, marker values were considered as the outcome variables and the tissue type, clinicopathologic and outcome features as the predictor variables. Variables were described using measurements of frequency and central tendency and dispersion. Comparison were visualized using barplots, boxplots, and density plots.

The association was evaluated using either the Mann-Whitney U test for the sum of ranks, the Kruskal-Wallis rank sum test, or the Pearson chi-squared test, depending on the features of the predictor variables.

 

Outcome analysis

The outcome analysis included regression modeling and time-to-event (survival) analysis. Markers levels were categorized as high/kow expression using the median as the cutoff point. Possible outcomes included the following:

  • NED: No evidence of disease
  • AWD: Alive with disease
  • DOC: Died from unrelated causes (i.e., non-cancer)
  • DOD: Died from disease (i.e., cancer)
  • LFU: Lost at follow-up

The outcome analysis included 3 endpoints: tumor progression, overall mortality (DOC + DOD vs. NED + AWD + LFU), and cancer-related mortality (DOD vs. NED + AWD + DOC + LFU). For all endpoints, total follow-up time was used.

Odds ratios were estimated using unconditional binary logistic regression. Hazard ratios were estimated using Cox’s proportional hazards regression. Survival curves were built using the Kapplan-Meier estimator and compared using the log-rank test.
